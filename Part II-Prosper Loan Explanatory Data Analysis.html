Investigation Overview¶

The goal of ther project was to invesigate on loan status and borrower-APR features, loooking at how they interact with each other and other categorical and numerical variables in the data set in order to get insights on relationships and trends. This will help to review key factors that affects loan status outcomes, and also explain important determinants for borrower's APR.

Dataset Overview¶

Prosper Marketplace, Inc. is a San Francisco, California-based company involved in the peer-to-peer lending industry. Prosper Funding LLC, one of its subsidiaries, operates Prosper.com, a website where individuals can either invest in personal loans or request to borrow money.The Propser loan data set contains 113,937 loan entries with 81 variables on each loan, including loan amount, borrower rate, current loan status, borrower income, and many others. This data dictionary elaborates on the features contained in the data set.

#group using credit lower score cut-offs
def credit_score(c):
    outcome=""
    if c <= 660 :
        outcome = "Poor"
    if c>=660:
        outcome = "Fair"
    if c>=680:
        outcome = "Good"
    if c>=720:
        outcome = "Excellent"
    return outcome
df_clean['credit_score'] = df_clean['credit_score_avg'].apply(credit_score)
